# epoch-MLW
epoch@MLW‘s project
## Random Forest tuning

The Random Forest workflow uses preprocessing artifacts generated by `preprocess.py` (saved under `processed/random_forest/`).

Run with fixed best parameters and generate a Kaggle submission:

```bash
python model_random_forest.py
```

Outputs:

- `submission_rf.csv`: Kaggle-ready submission with `PassengerId` and boolean `Transported`
- `outputs/models/random_forest_tuning_summary.json`: best CV accuracy and parameters

If you want to re-run hyperparameter tuning:

```bash
python -c "import model_random_forest as m; m.run(tune=True, n_iter=90)"
```

Tuning notes (Chinese): `report/random_forest_tuning_guide.md`
